## Algorithm

```
1) Start
2) Declare necessary variables
3) Declare array 'a' with 'n' elements.
4) Input number of elements 'n'
5) For i <-- to 'n'
	1) Input a[i]
	2) End for loop.
6) For i <-- 0 to n:
	1) For j <-- 0 to n:
		1) If (a[j] > a[j + 1]) Swap a[j] and a[j + 1]
		2) End j for loop
	2) End i for loop
7) For i <-- 0 to n;
	1) print a[i]
	2) End for loop
8) Stop.
```
